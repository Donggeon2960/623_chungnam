
# 📊 NumPy 기본 개념 정리

> NumPy(Numerical Python)는 파이썬에서 수치 계산을 위한 핵심 라이브러리입니다. 고성능의 다차원 배열 객체와 다양한 수학 함수들을 제공합니다.

---

## 📌 NumPy란?

- Python의 수치 계산을 위한 라이브러리
- 고성능 다차원 배열 객체인 `ndarray` 제공
- 선형대수, 푸리에 변환, 난수 생성 등 다양한 기능 포함
- 대부분의 과학/데이터 분석 패키지의 기반 라이브러리

---

## 🔢 NumPy 배열 (ndarray)

| 개념 | 설명 |
|------|------|
| `ndarray` | NumPy의 기본 데이터 구조, n차원 배열 |
| 차원(dimension) | 1차원 벡터, 2차원 행렬, n차원 텐서 |
| `shape` | 배열의 크기 (튜플 형태) |
| `dtype` | 배열의 데이터 타입 (int32, float64 등) |
| `size` | 전체 원소 수 |
| `ndim` | 배열의 차원 수 |

---

## 🛠 배열 생성 함수

| 함수 | 설명 |
|------|------|
| `np.array()` | 리스트/튜플을 배열로 변환 |
| `np.zeros()` | 0으로 채워진 배열 생성 |
| `np.ones()` | 1로 채워진 배열 생성 |
| `np.arange()` | 범위 지정 배열 생성 (range와 유사) |
| `np.linspace()` | 균등 간격의 숫자 배열 생성 |
| `np.eye()` | 단위 행렬 생성 |

---

## 🔁 배열 연산

| 항목 | 설명 |
|------|------|
| 브로드캐스팅 | 크기가 다른 배열 간 연산을 자동으로 확장 |
| 요소별 연산 | `+`, `-`, `*`, `/`, `**` 등 |
| 행렬 곱 | `@` 또는 `np.dot()` 사용 |
| 비교/논리 연산 | `>`, `<`, `==`, `logical_and`, `logical_or` 등 |

---

## 🔄 배열 조작

| 함수 | 설명 |
|------|------|
| `reshape()` | 배열 모양 변경 |
| `flatten()` | 다차원 배열을 1차원으로 |
| `transpose()` | 전치 (행과 열 바꾸기) |
| `concatenate()` | 배열 이어붙이기 |
| `split()` | 배열 나누기 |

---

## 🔍 인덱싱 & 슬라이싱

- 리스트와 유사한 방식으로 요소 접근 가능
- `:`를 이용한 범위 선택
- 불리언 인덱싱과 팬시 인덱싱 지원

---

## 📈 통계 함수

| 함수 | 설명 |
|------|------|
| `np.mean()` | 평균 |
| `np.median()` | 중앙값 |
| `np.std()` | 표준편차 |
| `np.sum()` | 합계 |
| `np.max()` / `np.min()` | 최댓값 / 최솟값 |

---

## 🎲 난수 생성

| 함수 | 설명 |
|------|------|
| `np.random.rand()` | 0~1 사이의 균일 분포 난수 |
| `np.random.randn()` | 정규 분포 난수 |
| `np.random.randint()` | 정수 난수 생성 |
| `np.random.seed()` | 난수 고정 (재현 가능성 확보) |
